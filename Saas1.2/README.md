### 微信SAAS^1.2

使用`react`的方式来构建界面

### 目录结构

```
├── README.md
├── package.json            # package.json
├── src                     # 组件和逻辑
└── webpack.config.js       # webpack配置文件
```


### 开发

```
克隆项目  git clone https://github.com/Zhou-Hepeng/saas.git
进入项目  cd Saas1.2
加载插件  npm install
启动服务  npm start
打包生产  npm run build:src
```
